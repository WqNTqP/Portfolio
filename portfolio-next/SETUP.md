# 📋 Quick Setup Guide

Get your portfolio running locally in 5 minutes!

## What You Need

1. **Node.js** (includes npm) - Download from [nodejs.org](https://nodejs.org)
2. **Git** - Download from [git-scm.com](https://git-scm.com)
3. **Code Editor** - VS Code from [code.visualstudio.com](https://code.visualstudio.com)

---

## Installation Steps

### 1️⃣ Open Terminal/PowerShell

```bash
cd c:\Users\Shadow\Desktop\School\Portfolio\portfolio-next
```

### 2️⃣ Install Dependencies

```bash
npm install
```

This downloads all required packages (takes 1-2 minutes).

### 3️⃣ Run Development Server

```bash
npm run dev
```

### 4️⃣ Open in Browser

Visit: **http://localhost:3000**

🎉 Your portfolio is running locally!

---

## Making Changes

1. Edit any file in the `components/` or `pages/` folders
2. Save the file - the browser automatically refreshes!
3. See your changes instantly
